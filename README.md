# FPGA-Based-Implementation-of-ASCON-Encryption-for-Secure-ECG-Signal-Transmission
Designed a SystemVerilog architecture on a PYNQ-Z2 FPGA to secure ECG transmission using ASCON encryption, FSM control, and UART communication. Vivado supported synthesis and hardware validation, while Python visualized ECG signals before/after encryption to verify secure real-time processing.
